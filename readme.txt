This game is to guess a word by guessing the letters used in it one by one. It will be played on the terminal.
The computer shows the user an encrypted version of a word.
The word will be randomly selected from a reference file, by first randomly selecting a line and then randomly selecting a word from
the line.
The user needs to guess the word by attempting to insert the letters of this word through the terminal.
In case the letter exists in the word it will be decrypted. Otherwise, the computer will prompt the user that a wrong choice is make
and they need to reattempt.
In case of three consecutive wrong answers the game will be over, and the user will lose.
At the end of each game the user will be commented if they would like to play again and in case the answer is no, the application 
will the exit.
